# media-scheduler-mac

一个用于教会媒体服侍排班的 macOS 小应用。

它帮助为每月主日安排 PPT 和周报服侍人员，并尽量自动避开常见冲突，适合固定成员的小团队长期使用。

## 功能

- 自动生成下月每个周日的排班
- 区分 PPT 与周报两组成员
- 支持“仅校对”成员
- 支持“暂停服侍”而不删除成员
- 避免同一主日同一人重复出现在多个岗位
- 限制成员连续两周服侍
- 限制成员每月最多 2 次服侍
- 支持生成后手动调整
- 支持复制排班图片
- 支持导出 PNG
- 支持打包为独立运行的 macOS App

## 技术栈

- Swift
- SwiftUI
- AppKit
- Swift Package Manager

## 本地运行

在项目目录执行：

```bash
swift run
```

## 打包应用

执行：

```bash
./package_app.sh
```

打包完成后，应用位于：

```bash
dist/media-scheduler-mac.app
```

## 数据说明

- 成员名单和排班数据保存在本机
- 当前使用 `UserDefaults` 进行本地存储
- 已暂停服侍的成员不会参与自动排班，但仍会保留在名单中

## 适用场景

适合：

- 教会主日媒体服侍排班
- 人数不多但需要稳定轮换的团队
- 希望保留固定名单、避免每次手动增删成员的使用场景

## 当前限制

- 当前仅支持 macOS
- 暂未提供自动化测试
- 暂未支持云同步或多人协作

## 后续可扩展方向

- 增加排班历史统计
- 支持导出 PDF 或表格格式

- 优化应用图标与界面视觉
